# ChatSphere Frontend

A modern real-time chat application frontend built with React.js. Users can register, login, create chat rooms, exchange messages in real time, and see typing indicators.

## Live Demo

https://chatsphere-real-time-chat-app.vercel.app

## Features

* User Registration & Login
* JWT Authentication
* Protected Routes
* Create Chat Rooms
* Real-Time Messaging
* Typing Indicators
* Responsive Design
* Mobile Friendly UI
* Multi-User Support
* WebSocket Integration (STOMP + SockJS)

## Tech Stack

* React.js
* JavaScript (ES6+)
* React Router DOM
* Axios
* SockJS
* STOMP.js
* HTML5
* CSS3
* Vite

## Installation

1. Clone the repository

git clone https://github.com/dileep1142/chatsphere-real-time-chat-app.git

2. Install dependencies

npm install

3. Configure environment variables

Create a .env file

VITE_API_URL=http://localhost:8080/api
VITE_WS_URL=http://localhost:8080/ws

4. Start development server

npm run dev
